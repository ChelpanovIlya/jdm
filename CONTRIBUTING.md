# Участие в проекте

Спасибо, что хотите улучшить этот промпт! Мы принимаем:
- Исправления логических ошибок в правилах
- Новые примеры моделей (`examples/input/` → `examples/output/`)
- Улучшения JSON Schema
- Документацию и переводы

## Как внести изменения
1. Форкните репозиторий и создайте ветку `feature/your-change`
2. Добавьте примеры или правьте `PROMPT.md`
3. Откройте Pull Request с описанием изменений

## Требования к PR
- Новые правила должны иметь примеры в `docs/`
- Сохраняйте обратную совместимость формата модели
